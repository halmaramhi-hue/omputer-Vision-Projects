# 🍾 Bottle Quality Inspection on a Production Line

## 📋 Overview
A computer vision project for automated quality inspection of bottles on a production line — detecting defects (e.g. cracks, misfills, cap issues, label misalignment) using image classification/detection.


## 🎯 Goal
- Automatically flag defective bottles from images captured on a production line
- Reduce manual inspection time and human error
